从远程库克隆
1.登录github，创建新仓库
New repositiry-填写Repository name，勾选 initialze this repository with a README-Creat repository
2.打开Git Bush，使用git clone克隆一个本地库
用到的指令：pwd 显示当前目录，cd 进入目录,mkdir 创建空目录,mkdir -p -p或--parents 若所要建立目录的上层目录目前尚未建立，则会一并建立上层目录；
Git Bush命令其实就是Linux命令，想深入了解Linux命令可访问： http://man.linuxde.net/
具体操作：
$ pwd
$ mkdir  -p E:/github/python
$ cd E:/github/python
$ pwd
$ git clone git@github.com:michaelliao/gitskills.git
$ git clone git@github.com:wenxuli2/python.git  记得要将wenxuli2/python换成你的账号名和仓库名
